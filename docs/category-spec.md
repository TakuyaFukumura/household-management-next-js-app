# カテゴリ仕様書

## 1. 概要

本ドキュメントは、家計簿アプリのカテゴリ体系の見直しおよび拡充に関する仕様を定義します。  
既存カテゴリの名称修正と、新規カテゴリの追加内容、定数定義の方針、サンプルデータの更新内容を記載します。

---

## 2. カテゴリ一覧

### 2.1 収入カテゴリ

| カテゴリ名 | 説明 |
|------------|------|
| 給与 | 会社からの月給・賞与など |
| 副業 | フリーランス・アルバイトなどの副収入 |

### 2.2 支出カテゴリ

#### 既存カテゴリ（名称変更あり）

| 変更前 | 変更後 | 変更理由 |
|--------|--------|----------|
| 食費 | 食料費 | 他のカテゴリ名称との統一（〜費の形式） |
| 娯楽 | 娯楽費 | 他のカテゴリ名称との統一（〜費の形式） |

#### 既存カテゴリ（変更なし）

| カテゴリ名 | 説明 |
|------------|------|
| 交通費 | 電車・バス・タクシー・ガソリン代など |
| 光熱費 | 電気・ガス・水道代など |
| 通信費 | スマートフォン・インターネット回線料金など |
| 日用品 | 洗剤・消耗品・生活雑貨など |
| 医療費 | 病院・薬局・医薬品など |

#### 新規追加カテゴリ

| カテゴリ名 | 説明 |
|------------|------|
| 住宅費 | 家賃・住宅ローン・管理費など |
| 投資額 | 株式・投資信託・積立NISAなどへの投資。一般的な家計簿には無いカテゴリだが、このアプリでは支出の１つとして扱う |
| 保険料 | 生命保険・医療保険・損害保険の保険料 |
| 特別費 | 冠婚葬祭・旅行・家電購入など一時的な大きな支出 |
| 交際費 | 飲み会・贈り物・会食など |
| 美容費 | 美容院・化粧品・エステなど |
| 教養費 | 書籍・セミナー・習い事・資格取得など |
| その他 | 上記いずれにも分類できない支出 |

## 3. カテゴリ定数の定義

カテゴリ名をアプリ内で一元管理するために、定数ファイルを新規作成します。

### 3.1 定数ファイルの配置

| ファイルパス | 説明 |
|-------------|------|
| `src/lib/constants.ts` | カテゴリ定数などアプリ全体で使用する定数を定義するファイル |

### 3.2 定数の設計例

```typescript
/** 収入カテゴリの一覧 */
export const INCOME_CATEGORIES = ['給与', '副業'] as const;

/** 支出カテゴリの一覧 */
export const EXPENSE_CATEGORIES = [
  '食料費',
  '住宅費',
  '光熱費',
  '通信費',
  '交通費',
  '日用品',
  '医療費',
  '娯楽費',
  '投資額',
  '保険料',
  '特別費',
  '交際費',
  '美容費',
  '教養費',
  'その他',
] as const;

/** 収入カテゴリの型 */
export type IncomeCategory = (typeof INCOME_CATEGORIES)[number];

/** 支出カテゴリの型 */
export type ExpenseCategory = (typeof EXPENSE_CATEGORIES)[number];
```
