# ダークモード機能 仕様書

## 1. 概要

本ドキュメントは、家計簿アプリへのダークモード切替機能追加に関する仕様を定義します。  
ユーザーが任意のタイミングでライトモードとダークモードを切り替えられるようにし、  
選択状態をブラウザのローカルストレージに保存することで、ページ再読み込み後も設定を維持します。

---

## 2. 要件

| No. | 要件 |
|-----|------|
| R-1 | ライトモードとダークモードを手動で切り替えられること |
| R-2 | 切替ボタンは共通ヘッダーの右端に配置すること |
| R-3 | ユーザーの選択はローカルストレージに保存し、ページ再読み込み後も維持されること |
| R-4 | 初回アクセス時はライトモードをデフォルトとすること |
| R-5 | 切替ボタンは現在のモードをアイコンで示すこと（ライトモード時は ☀️、ダークモード時は 🌙） |

---

## 3. UI 仕様

### 3.1 ヘッダーレイアウト

現在のヘッダー構造を以下のように変更します。

**変更前:**

```
| 家計簿アプリ                           |
```

**変更後:**

```
| 家計簿アプリ            [☀️ / 🌙ボタン] |
```

- ヘッダー左端: アプリタイトル「家計簿アプリ」（変更なし）
- ヘッダー右端: ダークモード切替ボタンを新規追加

### 3.2 切替ボタン仕様

| 項目 | 仕様 |
|------|------|
| 配置 | ヘッダー右端 |
| ライトモード時の表示 | ☀️（クリックするとダークモードへ切替） |
| ダークモード時の表示 | 🌙（クリックするとライトモードへ切替） |
| ボタン形状 | 角丸ボタン（`rounded-full`） |
| ボタンサイズ | 40×40px 程度 |
| ホバー時 | 背景色を変化させてインタラクティブ感を演出 |

## 4. 技術仕様

### 4.1 技術スタック

- **フレームワーク**: Next.js（App Router）
- **スタイリング**: Tailwind CSS 4（`dark:` バリアント利用）
- **状態管理**: React `useState` + `useEffect`
- **永続化**: `localStorage`（キー: `theme`、値: `"light"` / `"dark"`）

### 4.2 Tailwind CSS のダークモード設定

`globals.css` では既に以下のカスタムバリアントが定義されています。

```css
@custom-variant dark (&:where(.dark, .dark *));
```

`<html>` 要素に `dark` クラスを付与することでダークモードが有効になります。

### 4.3 状態管理フロー

```
初回ロード
  └─ localStorage に "theme" キーが存在するか？
       ├─ Yes → 保存済みの値を適用
       └─ No  → "light"（ライトモード）を適用

ボタンクリック
  └─ 現在のモードを反転
       ├─ "light" → "dark" に変更
       └─ "dark"  → "light" に変更
            ↓
       localStorage に保存
            ↓
       <html> 要素の "dark" クラスをトグル
```

### 4.4 ローカルストレージ仕様

| 項目 | 値 |
|------|----|
| キー | `theme` |
| ライトモード時の値 | `"light"` |
| ダークモード時の値 | `"dark"` |

## 6. コンポーネント設計

### 6.1 `Header.tsx`

ヘッダー全体を担うコンポーネントです。

```
Props（なし）
├─ アプリタイトル「家計簿アプリ」の表示
└─ DarkModeToggle コンポーネントの表示（右端）
```

**ダークモード対応スタイル例:**

```
ライトモード: bg-white  text-gray-800
ダークモード: bg-gray-900 text-gray-100
```

### 6.2 `DarkModeToggle.tsx`

ダークモードの状態管理と切替ボタンの表示を担うコンポーネントです。

```
State
└─ isDark: boolean  ← ダークモードかどうか

処理
├─ 初期化: localStorage から theme を読み込み、<html> に dark クラスを付与
├─ toggleTheme(): isDark を反転し、<html> の dark クラスをトグル、localStorage に保存
└─ レンダリング: isDark が true なら 🌙 アイコン、false なら ☀️ アイコンのボタンを表示
```
